## Getting Started

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/asearer/ecommerce-prototype.git
   ```

2. **Install Dependencies**:
   ```bash
   cd ecommerce-prototype
   npm install
   ```

3. **Start the Development Server**:
   ```bash
   npm start
   ```

4. **Visit** `http://localhost:3000` to view the app.
